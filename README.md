# Fatbook

Small app to track calories for personal use.

## Using Vite

Run in development mode:

```bash
npm run dev
```

Prod build:

```bash
npm run build
```

## Infra

* **Cloudflare Pages** - web hosting
* **Supabase** - DB, Auth
* **Google** - Auth Provider

## Check bundle size

```bash
npx vite-bundle-visualizer
```
