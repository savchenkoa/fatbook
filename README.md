# Fatbook

Small app to track calories for personal use.

## Using Vite

Install dependencies:

```bash
pnpm i
```

Run in development mode:

```bash
pnpm dev
```

Prod build:

```bash
pnpm build
```

## Infra

* **Cloudflare Pages** - web hosting
* **Supabase** - DB, Auth
* **Google** - Auth Provider

## Check bundle size

```bash
npx vite-bundle-visualizer
```
